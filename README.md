# msba-portfolio-samuelibikunle
Analyzing Trusource historical customer data to identify behavior patterns associated with churning. To help reduce customer churn we built a predictive model based on customer characteristics, usage, and company offerings. This will allow the company to focus on retention efforts and strengthen long term profitability
3 key results
Identified the strongest churn signals and high-risk customer segments by analyzing historical behavior, customer characteristics, and product/plan features.

Built a churn-prediction approach that enables early detection so retention efforts can be targeted before customers leave.

Produced actionable insights to improve customer retentions.
